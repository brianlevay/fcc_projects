URL Shortener API
How to Use this API This API takes a query in the form of a URL, and it returns a path from this site that re-directs to the original location. To generate a new link via URL, append: /api/new?url=path (path is the full url of the site you want) To visit the linked website, append: /api/handle (handle is your short character set) Examples of appropriate query formats: https://www.google.com, http://codepen.io, https://example.com Examples of URLs that won't work: www.mysite.com (missing http://), http://gohere (missing extension)
