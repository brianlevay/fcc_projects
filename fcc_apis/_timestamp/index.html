Timestamp API
How to Use this API This API takes a date in the form of either a Unix timestamp (an integer that gives the number of seconds passed since January 1, 1970), or a natural language date, and it returns a JSON object with both values. Examples of Unix timestamps: 1000000, 135420, -900000, etc. Examples of natural language dates: "February 2, 2015", "12 Mar 1999", "Oct-31-1916", etc. Examples of dates that won't work: "04/12/2001" - forward slashes not accepted, plus ambiguous day-month distinction
"July 2nd, 2007" - the "nd" on the end of the number will flag the day as a word To use the API via url, append the following: /api/?date=100000 - to get a natural date from a Unix timestamp
/api/?date=January 2, 2013 - to get a Unix timestamp from a natural date
