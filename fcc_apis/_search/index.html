Image Search API
How to Use this API This API uses Flickr to search for images. In addition, it can be used to get a list of the 10 most recent searches. To perform a new search via URL, append: /api/new?text=searchterms&?offset=pagenum

Enter your search terms after "text=". Offset is an optional query parameter, and it represents the page number of the results that you want returned. The API only gets the first 100 results from Flickr, and the results are broken into pages with a maximum of 20 images. To see the most recent searches, append: /api/recent/
API Test Lab You can also test the API by entering your search terms directly into the box below, or by clicking one of the buttons. You do not need to use the '/api/new?text=' prefix.
