Free Code Camp API Projects
About This Web App This app is a central portal to the five back-end API projects that are part of the Free Code Camp curriculum. You can click the links below to take you to the individual projects. The code for this app and all API endpoints can be found on GitHub.
API Projects Timestamp Conversion: Generates both UNIX and natural dates for a variety of time inputs Header Information: Returns information about the client browser based on request headers URL Shortener: Returns a short URL handle that can be used to redirect to another website Image Search Service: Returns photo search results from Flickr File Size Checker: Either uploads a file and returns its size, or determines the size on the client side
Created by Brian LeVay, July 2016 as part of the FreeCodeCamp curriculum
